<p align="center">
  PurgaLib is an advanced library/framework for SCP:SL that adds extended server features, custom role management, extended events, and automatic systems like the built-in updater.
</p>

---

<h2 align="center">ðŸ“– Description</h2>

<p>
PurgaLib provides:
<ul>
<li>Custom role system and permission management</li>
<li>Extended events for players, rounds, doors, elevators, Tesla interactions, etc.</li>
<li>API for plugin integration</li>
<li>Automatic updater via GitHub Releases</li>
<li>Cross-platform server support (Windows/Linux)</li>
<li>Verified player recognition and automatic credits system</li>
</ul>
